## About Proyect

Tomar un archivo Excel leerlo y extraer la información de ciertas celdas automáticamente para llevarlo a una data table en HTML que indique la información importante, además de permitir según el estado de guía de rastreo(solo guías en proceso, a guías en estado terminado no mostrar sincronización)  mostrar la fecha en que fue consultado su estado y sincronizar hasta que el proceso de entrega este terminado.
